§0. 동기 — 서비스 분할(R4-3) 전 독립 dedup

R4-1 이 BIOS 컨트롤러를 6갈래(Metadata/Lifecycle/Upload/Nudge/Integrity/Browse + Support)로 분리했다. 다음은 R4-3 의 서비스 분할(fat BiosService → Lifecycle/Upload/Integrity/Metadata, R3-3 미러링)이지만, 그 전에 분할을 깔끔하게 만드는 작은 독립 dedup 을 먼저 커밋한다 — BIOS 등록 3 경로에 marker 4-step 블록이 복붙돼 있어, 분할 시 어느 새 서비스로 가든 이 복붙이 따라다니기 때문이다. 먼저 BiosMarkerWriter 로 접어두면 R4-3 분할이 이 한 컴포넌트만 이동시키면 된다.

R4 시퀀스 — R4-1(컨트롤러 분리) → R4-2(marker dedup, 본 슬라이스) → R4-3(서비스 분할) → R4-4(cascade) → R4-5(@Lazy 제거, 이미 완료). 본 슬라이스는 가장 작은 dedup 으로 R4-3 의 사전 정지작업.

§2. 요구사항 (확정) + 비목표
  1. BiosMarkerWriter 추출 — management/bios/service/BiosMarkerWriter (@Component, ProvisionMarkerService 주입). 메서드 void writeSignedMarker(BoardBIOS bios, Path targetDir, Long boardId, String version, String entrypoint, String manifestHash) — BIOS attributes 조립 + ①~④ 4-step 을 내부 캡슐화.
  2. 3 call site 단일화 — addBios / registerExisting / persistFromNudge 의 4-step 블록을 biosMarkerWriter.writeSignedMarker(saved, targetDir, boardId, ver, entrypoint, manifestHash) 한 줄로. signed/unsigned.withSignature 표기 불일치 자연 소멸.
  3. 동작·서명·HMAC 100% 보존(불가침) — MarkerContent 필드(resourceType / resourceId=biosId / attributes / createdAt / manifestHash / signature) 동일, 2-phase save 순서(엔티티 선저장 → biosId → 서명 → reissue → write) 동일. 서명 입력 바이트 불변 → 기존 마커 검증 회귀 없음.
비 목표 (다음 슬라이스)
  • 서비스 분할 — fat BiosService → Lifecycle/Upload/Integrity/Metadata 분할은 R4-3. 본 슬라이스는 BiosService 내부 dedup 만(분할 아님). BiosMarkerWriter 는 R4-3 에서 Upload 서비스가 그대로 주입해 재사용.
  • 타 도메인 — BMC/Subprogram 의 동종 marker 복붙은 R5/R6 가 각자 BmcMarkerWriter 등으로(횡단 일반화는 over-abstraction 보류 — 도메인별 attributes 상이).
  • typed-name verifier 위임 / LifecycleService 구현 — R1-5/R3-5 의 SSOT 패턴(purgeWithTypedNameCheck inline → verifier)과 implements LifecycleService 는 R4-3/R4-4. 본 슬라이스 범위 밖.

@Component @RequiredArgsConstructor
public class BiosMarkerWriter {
  private final ProvisionMarkerService markerService;

  /** biosId 포함 unsigned 조립 → 서명 → 2-phase reissue → IN_TREE write. */
  public void writeSignedMarker(BoardBIOS bios, Path targetDir,
        Long boardId, String version, String entrypoint, String manifestHash) {
    MarkerContent unsigned = new MarkerContent(
        ResourceType.BIOS_BUNDLE.name(), bios.getId(),
        Map.of("boardId", String.valueOf(boardId),
               "version", version, "entrypointRelativePath", entrypoint),
        Instant.now(), manifestHash, null);
    String signature = markerService.computeSignature(unsigned);
    bios.reissueMarker(manifestHash, signature);
    markerService.write(targetDir, MarkerLayout.IN_TREE,
                        unsigned.withSignature(signature));
  }
}
