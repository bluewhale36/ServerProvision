§0. 동기 — fat 단일 컨트롤러의 책임 혼재 정리

현재 BmcController 는 409줄 · 24 엔드포인트 한 클래스에 조회 / 메타데이터 편집 / lifecycle(toggle·delete·restore·deprecate·purge) / 업로드(intent·upload·register-existing·form) / nudge(6) / job(verify·integrity) / browse 7 기능군이 뒤섞여 있다. 의존 필드만 8개(BmcService·BmcUploadIntentService·BmcNudgeService·BoardModelService·BmcVerificationLauncher·DirectoryBrowseService·DeleteIntentRegistry·TypedNameVerifier)로, 한 컨트롤러가 떠안기엔 책임이 과적이다.

이 분할은 OS 도메인 R1 의 8 컨트롤러 분할, BIOS 도메인 R4-1 의 동형 분할 선례를 BMC 에 미러링하는 작업이다. BMC 는 BIOS 와 동형(treeRootPath + IN_TREE 마커 번들)이므로 분배 경계가 거의 1:1 대응한다.

CLAUDE.md §조건 분기문 legacy 무분별 확장 경계 정합 — 본 프로젝트는 컨트롤러 try/catch 를 신규 컨트롤러에 복붙으로 번지게 한 회귀(S3.3→S3.4 4 컨트롤러 multi-catch 누락)를 반복했다. R5-1 은 분할 과정에서 BMC 의 잔존 try/catch 2곳(uploadBundle multi-catch · browse 4-catch)을 @RestControllerAdvice 로 승급해 분할이 곧 분기 제거가 되도록 한다.
§1. 현재 상태 (진단)

선행 완료 / 진입 전제

  • 완료 BMC 도메인 기능 자체(MA4)는 완성 — 본 슬라이스는 동작을 보존한 채 컨트롤러 구조만 정돈.
  • 선례 OS R1 8 컨트롤러 분할 · BIOS R4-1 동형 분할 — 같은 분배 경계 / Support 헬퍼 추출 패턴을 그대로 미러.
  • 선례 OS / BIOS browse 의 try/catch → @RestControllerAdvice 승급 작업이 선행되어 있어, BMC 가 합류하면 3 도메인 단일 advice 로 수렴.

§2. 요구사항 (확정) + 비목표
  1. URL · 동작 100% 불변 (불가침) — 24 엔드포인트의 Method / URL / 응답 status / 응답 바디 / produces 속성 전부 동일. 순수 controller 재배치(뷰 변경 0, Service 변경 0).
  2. 기능군 6 컨트롤러 분배 — BmcMetadataController(list·edit form·update) · BmcLifecycleController(toggle·delete·restore·deprecate·purge) · BmcUploadController(new form·intent·upload·register-existing) · BmcNudgeController(nudge×3 + intent-nudge×3) · BmcJobController(verify·integrity-status) · BmcBrowseController(browse).
  3. try/catch 2곳 → advice 승급 — uploadBundle multi-catch / browse 4-catch 를 @RestControllerAdvice 로 끌어올려 OS·BIOS browse advice 와 3 도메인 단일화. 분할 = 분기 제거.
  4. 복붙 헬퍼 Support 추출 — redirectToListWithSelect / nullToEmpty / populateFormContext + bindingResult 첫-에러 추출 + typedNameVerifier guard 를 BmcControllerSupport 정적 헬퍼로. N 컨트롤러 공유로 복붙 제거.
  5. AJAX formCard produces 보존 (불가침) — newForm 의 X-Requested-With 분기 + "... :: formCard" fragment 반환 경로를 그대로 보존. 분할 시 라우팅 회귀 가능 지점.
