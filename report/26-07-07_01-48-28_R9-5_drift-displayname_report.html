§4. ⚠ DDL 적용 — 사용자 필수 실행 (앱 기동 선행 조건)
ddl-auto=validate 이므로 아래 1줄을 적용하기 전에는 앱이 기동되지 않는다 (Schema-validation: missing column [display_name] in table [drift]). claude_code 계정은 ALTER 권한이 없음을 실측 확인(ERROR 1142)했고 본 세션은 root 자격 증명이 없어 적용을 대행할 수 없었다. ALTER 권한 계정(root 등)으로: 
mysql -u root -p server_provision < sql/R9-5_drift_display_name.sql
# 확인: mysql -u root -p server_provision -e "SHOW CREATE TABLE drift" | grep display_name

R9-1~R9-4 만 먼저 확인하고 싶다면: Drift.java/DriftResponse.java/PathReconciliationService.java(R9-5 diff)/list.html 의 R9-5 표시부를 되돌리는 방법도 있으나, DDL 1줄 적용이 훨씬 간단하다(nullable 추가라 무중단·무손실, 되돌리려면 DROP COLUMN).
