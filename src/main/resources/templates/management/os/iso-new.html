대상 OS :
서버 파일시스템상의 저장 경로. 직접 타이핑하거나 탐색 버튼으로 디렉토리 / 기존 ISO 파일을 골라 채울 수 있습니다. 업로드 시 이 경로에 직접 저장하며, 파일을 비우면 이 경로의 기존 ISO 를 그대로 등록합니다.
이 디렉토리에 저장: 현재 디렉토리를 ISO 경로로 적용 — 파일 업로드 시 multipart 파일명이 자동 추가됩니다.
대상 경로의 상위(부모) 디렉토리가 없을 때 함께 생성합니다. 대상 디렉토리 자체는 업로드 시 생성됩니다.
비워두면 파일 전송 없이 등록합니다 — 단, 서버가 위 경로의 기존 ISO 파일을 읽어 manifest 해시를 계산하고 .provision.json 마커를 발급합니다. 12GB 같은 큰 ISO 는 수 분이 걸릴 수 있습니다.
취소
등록이 완료될 때까지 페이지를 떠나지 마세요. 파일 업로드뿐 아니라 경로만 등록하는 경우에도 서버가 manifest 해시 계산과 마커 발급을 수행합니다. 완료 시 자동으로 목록 페이지로 이동합니다.
