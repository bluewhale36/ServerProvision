# Repository Guidelines

이 파일은 안내를 직접 담지 않고 기준 문서를 가리킨다. 과거에 다른 문서의 사본을 담았다가 낡은 채 남았던 적이 있어 포인터로만 유지한다.

| 알고 싶은 것 | 보는 곳 |
|---|---|
| 프로젝트 개요, 저장소 구조, 빌드와 실행 | `README.md` (환경변수 없이는 앱이 기동하지 않는다) |
| 개발 프로세스와 아키텍처 원칙 | `CLAUDE.md` |
| 문서 전체 지도 | `docs/README.md` |
| 화면 디자인 시스템 | `DESIGN.md` |
| 설정 항목과 기본값 | `src/main/resources/application.properties` 주석 |
| 작업 단위별 상태와 일정 | Notion DB 'Provisioning Server 개발 상세' (저장소 밖) |
| 엔티티와 도메인 상세 | 코드가 기준이다. 문서에 중복 기술하지 않는다 |

주의: `archive/legacy/`는 이전 구현의 참조용 사본이다. 실행 경로가 아니며 그 스타일과 관례를 따르지 않는다.
